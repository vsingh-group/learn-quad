### Requirements

```bash
# Clone the repository
git clone https://github.com/vsingh-group/learn-quad.git
cd learn-quad

# Install dependencies
pip install -r req.txt
```

### Dependencies

The main dependencies include:
- JAX 
- NumPy
- SciPy
- Matplotlib (for visualization)
